# SHORT BIBLE VERSE 
Is a simple package containing a stack of  shortest verses of the Holy Bible and generates them randomly.

## USAGE

Use the npm install

```bash
npm i short-bible-verse
```

## Usage

```javascript

const short_verse = require('short-bible-verse');

console.log(short_verse());


```
